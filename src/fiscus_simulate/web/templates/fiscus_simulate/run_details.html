{% endblock %} {% block glossary %}

Percentile & scenario. Scenarios are ranked by terminal net worth (nominal net worth at the final quarter). The percentile input selects the rank round(p/100 × (S−1)); the list shows that scenario and its near neighbors. Each option's years funded is the full horizon if the plan never breaks, else the first-shortfall quarter ÷ 4.

Reproducing a scenario. Runs don't store the per-scenario cube; the chosen scenario is regenerated bit-for-bit from the run's stored seed (the return draw is sequential, so scenario i is the i-th block of draws). The engine reruns for that lone path, recording balances every quarter.

Consolidated walk. Per quarter, summed over the taxable, tax-deferred and tax-free accounts. BOP Stocks/Bonds/Cash/Total = beginning-of- period balances by asset (the ending equals the next row's beginning, so it is not repeated). Expense = funded spending (below plan only when assets are exhausted); Income = pensions/other external income + dividends & interest; Savings = pre-retirement contributions; Realized G/L = capital gains crystallized by sales; Tax = income tax + sale tax; Δ Unrealized = the period's capital return less the realized part (change in paper gains); Total change = End − BOP Total. The identity Total change = Income + Savings − Expense − Tax + Realized G/L + Δ Unrealized holds every row. Timing is life-actuarial: spending is paid at the start of the quarter from cash; investment income, capital return and pensions land at the end; ordered sales (taxable → tax-deferred → tax-free) and RMDs settle there.

By account. The same walk split across the taxable, tax-deferred and tax-free accounts (plus a Total): beginning Stocks/Bonds/Cash, investment Income, Realized gains (RG — only the taxable account crystallizes capital gains; tax-deferred is taxed as ordinary income on withdrawal) and the change in Unrealized gains (ΔUnrl).

Spending is never cut. V1 keeps spending on the plan even once assets are exhausted — the shortfall is funded by debt (net worth goes negative), so ruin is exposed at its true depth rather than floored at zero. "Years funded" is the time until net worth first goes negative.

Funnel overlay. The pink line is this scenario's own net worth by quarter, drawn over the population bands (p10–p90 outer, p30–p70 inner). It can wander outside the bands — a percentile band is a per-quarter cross-section, not any single household's path. Real divides each quarter by the inflation deflator (1+π)t.

Order of returns. This scenario's 160 quarterly return vectors are permuted (no replacement) n times, identically across stocks/bonds/cash so within-quarter correlation is preserved; inflation is constant in V1, so it is unaffected. The drawdown is rerun for each ordering from the same starting portfolio, isolating sequence-of-returns risk from the return environment. The histogram is terminal net worth across reorderings; Actual (this order) marks the realized scenario. A seed makes it reproducible; left blank it is random. This is a throwaway preview — not saved with the run.
